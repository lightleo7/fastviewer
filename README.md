# Rutube MPV FastViewer

Десктопный клиент для мгновенного поиска видео и плейлистов на Rutube с последующим стримингом потока напрямую во внешний плеер **MPV**. Приложение написано на Python с использованием современного GUI-фреймворка **Flet (Flutter под капотом)** и спроектировано по стандарту `src layout`.

---

## ✨ Особенности проекта

*   **Высокая скорость работы:** Никакого тяжелого парсинга HTML через Selenium. Клиент напрямую интегрирован с комбинированным внутренним API Rutube.
*   **Стриминг в MPV:** Запуск воспроизведения в один клик без рекламы и тормозов браузера. Основной интерфейс приложения при этом остается отзывчивым.
*   **Умное разделение контента:** 
    *   Автоматически определяет тип контента (**ВИДЕО** или **ПЛЕЙЛИСТ**).
    *   Для плейлистов генерирует корректные ссылки вида `https://rutube.ru/plst/{id}`.
*   **Современный UI:** Адаптивные горизонтальные карточки в темной теме Material Design 3 с отображением обложки, названия ролика и имени автора (канала).
*   **Асинхронная архитектура:** Полностью неблокирующий ввод/вывод (Asyncio + Httpx).

## 🛠 Системные требования и зависимости

Для работы приложения необходим **Python 3.11+**, а также установленный менеджер пакетов **uv**.

Основные библиотеки (устанавливаются автоматически):
*   `flet >= 0.85.0` (современный декларативный GUI)
*   `httpx` (асинхронный HTTP-клиент)
*   `yt-dlp` (консольная утилита для вытягивания потокового видео, необходима для MPV)

## 🚀 Быстрый запуск

### 1. Клонирование и подготовка
Перейдите в директорию с проектом и убедитесь, что у вас установлен менеджер `uv`. Если его нет, установите одной командой:
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh | iex"
```

### 2. Настройка плеера MPV (Обязательно)
Чтобы приложение могло воспроизводить видео, операционной системе нужен доступ к плееру `mpv` и утилите `yt-dlp`.
1. Скачайте свежую сборку плеера для вашей ОС (например, с официального сайта [mpv.io](https://mpv.io)).
2. Положите файл **`mpv.exe`** прямо в корневую папку проекта `fastviewer/` (рядом с файлом `pyproject.toml`).
3. Приложение автоматически подтянет `yt-dlp` внутрь изолированного виртуального окружения при первом запуске.

### 3. Запуск приложения
Синхронизация окружения и запуск осуществляются одной командой из корня проекта:
```bash
uv run python -m src.main
```

## 💻 Краткое описание кодовой базы

*   **`src/main.py`** — Стартовый файл. Корректирует пути поиска модулей Python для работы структуры `src` и запускает цикл приложения `ft.run()`.
*   **`src/parser.py`** — Создает асинхронную сессию `httpx.AsyncClient`. Сначала имитирует посещение главной страницы Rutube для получения базовых cookies, после чего отправляет поисковый запрос к эндпоинту `api/search/combined/video_playlist`. Формирует чистый массив данных.
*   **`src/ui.py`** — Описывает разметку окна (TextField, GridView/ListView, Card). Содержит асинхронный метод `play_in_mpv`, который через `asyncio.create_subprocess_exec` вызывает плеер, параллельно отображая системные уведомления `ft.SnackBar` через `page.overlay`.
